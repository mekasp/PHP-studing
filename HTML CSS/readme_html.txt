HTML-это язык гипертекстовой разметки,который применяется для отображения в браузерах
<!DOCTYPE html> -говорит как браузеру воспроизводить документ
<html> - основной тег
    <head> - настройки документа
      <title></title> - титулка документа,отображаемая во вкладке
      <link /> - ccылка на подключаемый документ(css или шрифты)
      <script /> - элемент для подключения .js файлов
      <script></script> - для написания .js кода
      <style></style> - внутри него пишутся стили (css)
      <meta /> - текстовая информация о сайте для поисковиков,а также настройки документа
    </head>

    <body> - визуал документа
       <div></div> - блочный элемент
       <a href="ссылка">текст</a> - ссылка
       <p></p> - параграф
       <h1></h1> (h2,h3,h4,h5) - заголовки
       <b></b> <strong></strong> - жирный текст
       <i></i> - иконки
       <span></span> - текст
       <br /> -перенос строки
       <img src="" /> - картинка
       <script></script - скрипты
       <form> - формы
          <input type="text,email,password,checkbox,hidden,submit" name="" value="" /> - поле формы
          <textarea name=""> какой-то текст <textarea> - поле для ввода описания
          <select> - поле для выбора опции
            <option>Russia</option>
            <option>Russia</option>
          </select>
          <button type="submit,button"></button> - кнопка
          <table> - таблицы
            <thead> - шапка
              <tr> -строчка таблицы
                <th></th> - столбец с жирным шрифтом
              </tr>
            </thead>
            <tbody> - данные
               <tr> -строчка таблицы
                <tв></tв> - столбец с обычным шрифтом
                </tr>
            </tbody>
          </table>

       </form>

    </body>
<html>
